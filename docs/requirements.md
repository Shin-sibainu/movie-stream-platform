# 動画配信プラットフォーム 要件定義書

## プロジェクト概要

Udemy のような動画配信プラットフォームのデモ版を開発します。月額制での動画配信を想定していますが、デモ版では認証と決済機能は実装せず、誰でも無料で動画を閲覧できるようにします。

## 機能要件

### 1. トップページ（コース一覧）

- **目的**: ユーザーが利用可能なコースを一覧で確認できる
- **機能**:
  - 複数のコースをカード形式またはグリッド形式で表示
  - 各コースカードには以下の情報を表示:
    - コースタイトル
    - コースのサムネイル画像
    - コースの概要（簡易版）
  - コースカードをクリックすると、そのコースの詳細ページに遷移

### 2. コース詳細ページ

- **目的**: ユーザーがコースの詳細情報を確認し、学習を開始できる
- **機能**:
  - コースの詳細情報を表示:
    - コースタイトル
    - コースの説明・概要
    - コースのサムネイル画像
    - セクション一覧（セクション名のみ）
  - 「コースを見る」ボタンを配置
  - 「コースを見る」ボタンをクリックすると、動画視聴ページに遷移

### 3. 動画視聴ページ（コース学習ページ）

- **目的**: ユーザーがセクションごとに動画を順番に視聴できる
- **機能**:
  - **セクション構造**:
    - コースは複数のセクションに分かれている
    - 各セクションには複数の動画が含まれる
  - **動画プレイヤー**:
    - YouTube 埋め込みプレイヤーを使用
    - 現在選択されている動画を再生
  - **動画リスト**:
    - 左側または下部にセクションと動画のリストを表示
    - セクションごとにグループ化
    - 各動画のタイトルを表示
    - 現在視聴中の動画をハイライト表示
  - **動画遷移機能**:
    - 一つの動画を見終わったら、次の動画に自動的に遷移（または手動で次の動画ボタンを表示）
    - セクション内の動画を順番に視聴できる
    - セクション間の移動も可能

### 4. ナビゲーション

- **目的**: ユーザーがスムーズにアプリ内を移動できる
- **機能**:
  - トップページへの戻るボタンまたはリンク
  - コース詳細ページへの戻るボタンまたはリンク
  - ブラウザの戻るボタンでも適切に動作

## 技術要件

### 動画配信

- **動画の埋め込み**: YouTube 埋め込みプレイヤーを使用
- **動画データ**: 各コース、セクション、動画の情報はデータベースまたは JSON ファイルで管理

### データ構造

```
コース (Course)
├── コースID
├── タイトル
├── 説明
├── サムネイル画像URL
└── セクション (Section[])
    ├── セクションID
    ├── セクション名
    └── 動画 (Video[])
        ├── 動画ID
        ├── タイトル
        ├── YouTube動画ID
        └── 説明（オプション）
```

## 非機能要件

### 認証・決済

- **認証機能**: 実装しない（デモ版のため）
- **決済機能**: 実装しない（デモ版のため）
- **アクセス制限**: なし（誰でも無料で全コース・全動画にアクセス可能）

### パフォーマンス

- ページ遷移はスムーズに動作すること
- YouTube 埋め込みプレイヤーが適切に読み込まれること

### UI/UX

- Udemy のような直感的な UI/UX を目指す
- レスポンシブデザイン（モバイル・タブレット・デスクトップ対応）

## 実装優先順位

### Phase 1（MVP）

1. トップページ（コース一覧）の実装
2. コース詳細ページの実装
3. 動画視聴ページの基本実装（セクション・動画リスト表示、YouTube 埋め込み）

### Phase 2（機能拡張）

1. 動画の自動遷移機能
2. 視聴進捗の表示（視聴済み動画のマーキング）
3. UI/UX の改善

### Phase 3（将来の拡張）

1. 認証機能の追加
2. 決済機能の追加
3. ユーザーごとの進捗管理
4. コメント機能
5. お気に入り機能

## 備考

- デモ版として開発するため、認証・決済機能は後回しにする
- 将来的に月額制の動画配信プラットフォームとして拡張することを想定
- 動画は YouTube 埋め込みのため、YouTube の利用規約に準拠する必要がある
