## プロジェクト概要

Udemyのような動画配信プラットフォームのデモ版です。月額制での動画配信を想定していますが、デモ版では認証と決済機能は実装せず、誰でも無料で動画を閲覧できるようにします。

### 主な機能

1. **トップページ（コース一覧）**: 複数のコースを一覧表示
2. **コース詳細ページ**: コースの詳細情報を表示
3. **動画視聴ページ**: セクションごとに動画を順番に視聴

### 技術スタック（予定）

- Next.js
- TypeScript
- Tailwind CSS
- YouTube埋め込みプレイヤー

詳細は [tech-stack.md](./tech-stack.md) を参照してください。
